Compile:
	Put the makefile in the same directory as the c files for each program.
	typ"make" to compile
	"make clean" will remove the executables and the generated .txt files

Run:
	From the directory the files were compiled:
		mergesort: "./mergesort"
			produces merge.txt
		insertsort: "./insertsort"
			produces insert.txt
		mergeTime: "./mergeTime"
			command line output
		insertTime: "./insertTime"
			command line output
